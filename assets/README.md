# Assets Directory

This directory contains images and other assets for the learning paths.

## Required Images

### Open Graph Images (for social sharing)

Create these images (1200x630px recommended):

1. **og-image.png** - Main landing page image
   - Should represent both learning tracks
   - Include project title and tagline
   - Use gradient backgrounds matching the design

2. **og-native.png** - AI-Native Developer track image
   - Blue/Indigo gradient theme
   - Include "AI-Native Developer" text
   - Show JavaScript/React/Node.js icons

3. **og-agent.png** - AI Agent Developer track image
   - Purple/Emerald gradient theme
   - Include "AI Agent Developer" text
   - Show AI/MCP/Agent icons

### Favicon

4. **favicon.ico** - Browser tab icon (16x16, 32x32, 48x48)
5. **favicon.png** - High-res version (512x512)

## Image Guidelines

- **Format**: PNG for transparency, JPG for photos
- **Size**: Optimize for web (use tools like TinyPNG)
- **Accessibility**: Include alt text in HTML
- **Branding**: Maintain consistent color scheme

## Tools for Creating Images

- **Canva**: Easy drag-and-drop design
- **Figma**: Professional design tool
- **GIMP**: Free Photoshop alternative
- **Photopea**: Online Photoshop alternative

## Temporary Solution

Until custom images are created, the meta tags reference placeholder paths. Social media platforms will use default previews.
