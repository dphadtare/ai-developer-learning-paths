## 📚 About

This repository contains two comprehensive, self-paced learning paths designed to transform developers into AI-powered professionals:

### 💻 AI-Native Developer Track
Master full-stack development with AI assistance. Build production-ready applications using modern JavaScript, APIs, databases, and React.

**Duration:** 6 weeks (30 days, Mon-Fri)  
**Time Commitment:** 2-3 hours per day  
**Tech Stack:** JavaScript, Node.js, Express, PostgreSQL, React, Docker

**What You'll Build:**
- RESTful APIs with authentication
- Database-driven applications
- Modern React UIs
- Production-ready deployments

### 🤖 AI Agent Developer Track
Build intelligent AI systems and master agent orchestration. Learn prompt engineering, tool development, MCP, and production AI deployment.

**Duration:** 6 weeks (30 days, Mon-Fri)  
**Time Commitment:** 2-3 hours per day  
**Tech Stack:** Python, MCP, LangChain, OpenAI, Claude, Cursor

**What You'll Build:**
- Custom AI tools and integrations
- MCP servers and clients
- Multi-agent orchestration systems
- Production AI workflows

## ✨ Features

- **📖 Structured Curriculum**: 6-week programs with daily milestones and progressive difficulty
- **🧪 Interactive Labs**: Hands-on coding exercises with real-world scenarios
- **✨ AI Mentor Integration**: Built-in Chrome Gemini integration for instant help
- **📊 Progress Tracking**: Track your completion with localStorage-based progress
- **🎯 Self-Paced**: Learn at your own speed with clear learning objectives
- **💾 Offline Support**: Download and use offline after initial load

## 🛠️ Prerequisites

### AI-Native Developer Track
- Basic understanding of programming concepts
- Familiarity with command line/terminal
- Text editor or IDE (VS Code recommended)
- Chrome browser (for AI mentor integration)

### AI Agent Developer Track
- Programming experience (Python preferred)
- Understanding of APIs and web services
- Familiarity with AI concepts (helpful but not required)
- Chrome browser (for AI mentor integration)

## 🎓 How to Use

1. **Choose Your Track**: Visit the [landing page](https://dphadtare.github.io/ai-developer-learning-paths/) and select your learning path
2. **Set Up Chrome Gemini**: Follow the [Setup Guide](docs/SETUP_GUIDE.md) to enable AI mentor integration
3. **Start Learning**: Begin with Week 1, Day 1 and progress through the curriculum
4. **Complete Labs**: Each day includes hands-on labs - complete them to reinforce learning
5. **Track Progress**: Use the built-in progress tracker to monitor your completion
6. **Get Help**: Use the "Ask AI Mentor" feature for instant guidance on any topic
